# minimum spanning tree (mst)

## what i did

i used kruskal algorithm to find the minimum spanning tree

each computer is a node
each cable is an edge with cost

## idea

- sort edges by cost
- pick smallest edge
- avoid cycles using union find

## result

selected edges give minimum total cost
and all nodes are connected

## complexity

O(E log E) بسبب sorting

## conclusion

kruskal is simple and works well for this problem
